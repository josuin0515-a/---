# 🔒 안전한 비밀번호 생성

비밀번호를 안전하게 생성하도록 돕는 웹 애플리케이션입니다.

## ✨ 주요 기능

- **비밀번호 강도 체크기**: 실시간 비밀번호 보안 강도 평가 및 안전한 랜덤 비밀번호 추천
- **조건 체크**: 8자 이상, 대문자 포함 (A-Z), 소문자 포함 (a-z), 숫자 포함 (0-9), 특수문자 포함 (!@#$%^&* 등)
- **정렬**: 체크 상태 기준 정렬 (미체크/체크)
- **조건 충족 시 화면 전환**: 8자 이상 + 다른 조건 2개 이상 만족 시 "비밀번호가 생성되었습니다" 표시
- **반응형 디자인**: 모바일/태블릿/데스크톱 지원

## 📁 파일 구조

## 🚀 사용 방법

1. 프로젝트 폴더에서 `index.html` 파일을 브라우저로 엽니다.
2. 비밀번호 입력창에 비밀번호를 입력하면 실시간 강도 평가가 표시됩니다.
3. **조건 체크** 에서 조건 충족 여부 확인 가능:
    -8자 이상
    -대문자 포함
    -소문자 포함
    -숫자 포함
    -특수문자 포함
4. "추천 비밀번호" 버튼을 클릭하여 안전한 랜덤 비밀번호 확인 및 선택 가능
5. "비밀번호 생성하기" 버튼 클릭 시 8자 이상 포함 + 다른 조건 2개 이상 만족할 때 화면 전환
6. 조건을 만족하지 않으면 경고 메시지 표시

## 🔐 비밀번호 생성기

### 평가 기준
- **8자 이상**: 최소 길이 충족
- **영문 대문자 포함**: A-Z
- **영문 소문자 포함**: a-z
- **숫자 포함**: 0-9
- **특수문자 포함**: !@#$%^&* 등

### 강도 등급
- **매우 약함** (0-1개 조건): ⚠️ 개선 필요
- **약함** (2개 조건): ⚠️ 더 강화 필요
- **보통** (3개 조건): ⚡ 양호
- **강함** (4개 조건): ✅ 안전
- **매우 강함** (5개 조건): ✅ 완벽 🎉

### 화면 전환 조건
- 8자 이상 포함 + 나머지 조건 중 2개 이상 만족 시
- 조건 미충족 시 알림창 표시

## 📝 체크리스트 항목

### 비밀번호 보안
- 서로 다른 사이트에 같은 비밀번호 사용하지 않기
- 복잡한 비밀번호 설정 (숫자, 대문자, 특수문자)
- 2단계 인증(2FA) 적용

### SNS 보안
- 위치정보 공개하지 않기
- 개인정보 공개 제한
- 프로필 공개 범위 제한

### 기기보안
- 잠금 패턴/PIN 설정
- 운영체제 및 앱 최신화
- 정기적 백업

### 인터넷 보안
- 공용 와이파이에서 민감정보 입력 금지
- SSL/TLS 암호화 사이트 사용
